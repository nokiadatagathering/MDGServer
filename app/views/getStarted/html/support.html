FREQUENTLY ASKED QUESTIONS

1) What are the benefits of mobile data collection?

The key benefits of mobile data collection are speed, accuracy, cost efficiency and usability. Please see the benefits section on the homepage for more on benefits.

2) How is Microsoft Data Gathering used in the field?

When time is of the essence, rapidly available, accurate information is critical for sound decision-making in any kind of organization.

For example: In Brazil, the Amazonas State Health Department and the Health Vigilance Foundation have used Microsoft Data Gathering to monitor and control the spread of dengue fever.

In East Africa, the Food and Agriculture Organization has conducted water and livestock surveys to reduce the risk of vulnerable communities to drought.

In Kenya, Plan Finland has worked with the Kenyan government to register births using Microsoft Data Gathering.

For more on use cases, please see the use cases page.

3) Does it cost anything to use Microsoft Data Gathering?

Nokia Data Gathering is an open source solution so there are no licensing fees or charges for using the software. For more on open source, please see the open source page.

Of course, there are costs associated with the hardware (mobile phones, servers, SIM cards) necessary for using Nokia Data Gathering, as well as the implementation of a data collection project. These costs must be scoped and borne by the organization using the solution.

4) What are the license terms for Microsoft Data Gathering?

The Microsoft Data Gathering Windows Phone client is licensed under the open source Berkeley Software Distribution (BSD) License family. The Microsoft Data Gathering Playground Server and the Java client are both licensed under the GNU Lesser General Public License (ILGPL).Microsoft Data Gathering complies with the standards of the OpenROSA Consortium on mobile data collection.

For more on our licenses, please see the open source page.

5) Which devices are compatible with Microsoft Data Gathering?

The Windows Phone version of the Microsft Data Gathering mobile client is a Silverlight application for Windows Phone 7 and a Cordova/Sensa Touch application for Windows Phone 8. The Windows Tablet version runs on Windows RT devices. The Java application runs on Symbian and Nokia OS devices.

If you have a specific query about device compatibility, please let us know by sending an email to support@nokiadatagathering.net.

6) Where can I find the Microsoft Data Gathering source code?

You can get the source code at https://github.com/nokiadatagathering.

7) How can I host my own server data?

The instructions for setting up and hosting your own server are contained in the README.

8) What are the previous versions of Microsoft Data Gathering?

Please see our release archive for more information on previous releases of Microsoft Data Gathering.

9) Can students work on Microsoft Data Gathering?

Yes. Please see the description of our University Collaboration Program.

10) Can I translate the Microsoft Data Gathering mobile client into my language?

Yes. Please get in touch with us at support@nokiadatagathering.net and work with us to provide a translation of the software in your language. We will then push this out and make the language version available to all users.
